###常量:

常量为全局的且使用时不带$符

系统常量:

| ———- |
| _FILE_ |php程序文件名 |
| _LINE_ | 当前代码行数|
| PHP_VERSION |当前解析器的版本号 |
| PHP_OS | 执行当前PHP版本的操作系统名称|

###函数:

| ———- |
| var_dump() | 显示关于一个或多个表达式的结构信息，包括表达式的类型与值。数组将递归展开值，通过缩进显示其结构。|
|fopen |fopen() 函数打开一个文件或 URL。如果 fopen() 失败，它将返回 FALSE 并附带错误信息|
|mixed constant(string constant_name)|获取常量值|
|bool defined(string constants_name)|判断常量是否被定义|
|unset() |用于销毁给定的变量|
|error_reporting() |设置当前脚本的错误报告级别(返回旧的错误报告级别)|
| memory_get_usage() | 获取内存消耗|
